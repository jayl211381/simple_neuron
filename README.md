Artificial neuron for linear regression
This is part of my Medium article on artificial neurons - "Artificial Neurons - And the question that started it all"

The Neuron class contains a simple implementation of an artificial neuron with a relu activation function and a MSE loss function. 

An example is provided under test/temperature_fitting.py to train a neuron to find a line of best fit for some sample temperature data. Run the code without changing any hyperparameters to replicate the results from the article. 

For further learning, I recommand Andrej Karpathy and 3Blue1Brown's videos on neural networks.
